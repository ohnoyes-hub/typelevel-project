# Jobsboard App

A fully functional Jobsboard application built with the Typelevel stack.

The app was previously hosted at [jobsboard.ohnoyes.com](https://jobs-staging.ohnoyes.com/), but hosting is temporarily disabled while migrating from DigitalOcean to a new VPS provider.

# Features

- Users can pay to post job ads via Stripe integration.
- User authentication (and password recovery currently in progress).
- Filtering and pagination for browsing job listings.
